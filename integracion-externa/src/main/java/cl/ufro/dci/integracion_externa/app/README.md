# App - Configuración de Integración Externa

## Descripción
Configuración principal y componentes de la aplicación de integración externa.

## Responsabilidades
- Configuración de beans y componentes principales
- Inicialización de servicios de integración
- Configuración de propiedades de aplicación

## Servicios Integrados
- SII (Servicio de Impuestos Internos)
- Transbank (Pasarela de pagos)
- Vademécum (Base de datos de medicamentos)
