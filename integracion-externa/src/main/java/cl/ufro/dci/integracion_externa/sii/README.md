# SII - Integración Servicio de Impuestos Internos

## Descripción
Módulo de integración con el Servicio de Impuestos Internos de Chile.

## Funcionalidades
- Validación de RUT y datos tributarios
- Emisión de documentos electrónicos (DTE)
- Consulta de estado de contribuyentes
- Envío y validación de boletas y facturas

## Endpoints Principales
- Autenticación SII
- Emisión de DTE
- Consulta de estado
