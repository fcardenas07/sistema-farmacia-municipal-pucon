# Vademécum - Integración Base de Datos de Medicamentos

## Descripción
Módulo de integración con bases de datos de medicamentos y vademécum.

## Funcionalidades
- Búsqueda de medicamentos por principio activo
- Consulta de información farmacéutica
- Validación de interacciones medicamentosas
- Sincronización de catálogos de medicamentos
- Consulta de precios de referencia

## Fuentes de Datos
- Base de datos nacional de medicamentos
- Vademécum electrónico
- Información de ISP
