# Transbank - Integración Pasarela de Pagos

## Descripción
Módulo de integración con Transbank para procesamiento de pagos.

## Funcionalidades
- Procesamiento de pagos con tarjeta de crédito/débito
- Transacciones OneClick
- Transacciones WebPay
- Anulación y reversa de transacciones
- Consulta de estado de transacciones

## Flujos Soportados
- WebPay Plus
- WebPay OneClick
- Transacción Completa
